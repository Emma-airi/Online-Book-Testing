Lecture 6#

Orbital Elements from Position and Velocity Vectors#

  1. Orbital Parameters from Position and Velocity Vectors

    • Calculate the semi-major axis \(a\) from the Vis-Viva equation:

      \[ a = \frac{\mu}{2\epsilon} \]

    • Compute the eccentricity \(e\) from:

      \[ \vec{e} = \frac{\vec{v} \times \vec{h}}{\mu} - \frac{\vec{r}}{|\mathbf{r}|} \]

    • Compute the inclination \(i\) from:

      \[ \mathbf{h_z}=\mathbf{h}\cos i \]

    • Compute the right ascension of the ascending node \(\Omega\) from:

      \[ \cos \Omega = \frac{n_y}{|\mathbf{n}|} \]

      Determine the correct quadrant:

      • If \(n_y \geq 0\), then \(\Omega \in [0, 180^\circ]\)

      • If \(n_y < 0\), then \(\Omega = 360^\circ - \cos^{-1}(n_x/|\mathbf{n}|)\)

    • Compute the argument of perigee \(\omega\) from:

      \[ \cos \omega = \frac{\mathbf{n} \cdot \mathbf{e}}{|\mathbf{n}| |\mathbf{e}|} \]

      Determine the correct quadrant:

      • If \(e_z \geq 0\), then \(\omega \in [0, 180^\circ]\)

      • If \(e_z < 0\), then \(\omega = 360^\circ - \cos^{-1}(\mathbf{n} \cdot \mathbf{e} / |\mathbf{n}||\mathbf{e}|)\)

    • Compute the true anomaly \(\theta\) from:

      \[ \vec{r}\cdot\vec{e} = |\mathbf{r}||\mathbf{e}|\cos \theta \]

      Determine the correct quadrant: Need to calculate \(v_r = \vec{v}\cdot\hat{r}\)

      • If $\(v_r \geq 0, \theta = \cos^{-1}(\frac{\vec{r}\cdot\vec{e}}{|\mathbf{r}||\mathbf{e}|}) \)$

      • If $\(v_r < 0, \theta = 360^\circ - \cos^{-1}(\mathbf{e} \cdot \mathbf{r} / |\mathbf{e}||\mathbf{r}|)\)$

    Note: Orbital parameters can be singular in certain conditions!

Rotation Matrix Elements:

  • Consider unit vectors \(\mathbf{I}\), \(\mathbf{J}\), and \(\mathbf{K}\) of a reference frame and their transformation to another frame \(\mathbf{I'}\), \(\mathbf{J'}\), \(\mathbf{K'}\):

    \[ \mathbf{F'} = A \mathbf{F} \]

    where \(A\) is the rotation matrix. Each column of \(A\) contains the components of \(\mathbf{I}\), \(\mathbf{J}\), \(\mathbf{K}\) in the new reference frame.

  • Inverse Transformation:

    • To compute the original vector from the transformed one:

      \[ \mathbf{F} = A^T \mathbf{F'} \]

      where \(A^T\) is the transpose of the rotation matrix \(A\).

  • Successive Rotations:

    • Given two successive rotations \(A_1\) and \(A_2\), the net rotation \(A\) is:

      \[ A = A_2 A_1 \]

    • Rotation about a Single Axis:

      • Rotation about \(\mathbf{I}\):

        \[\begin{split} A = \begin{bmatrix} 1 & 0 & 0 \\ 0 & \cos \theta & -\sin \theta \\ 0 & \sin \theta & \cos \theta \end{bmatrix} \end{split}\]

      • Rotation about \(\mathbf{J}\):

        \[\begin{split} A = \begin{bmatrix} \cos \theta & 0 & \sin \theta \\ 0 & 1 & 0 \\ -\sin \theta & 0 & \cos \theta \end{bmatrix} \end{split}\]

      • Rotation about \(\mathbf{K}\): $\( A = \begin{bmatrix} \cos \theta & -\sin \theta & 0 \\ \sin \theta & \cos \theta & 0 \\ 0 & 0 & 1 \end{bmatrix} \)$

ECI to Perifocal Frame#

  • Transformation:

    • Passage from the ECI frame to the perifocal frame is given by the composition of three rotations:

      \[ \mathbf{F_{perifocal}} = A_3(\omega) A_2(i) A_1(\Omega) \mathbf{F_{ECI}} \]

    • Inverse Transformation:

      • To transform from the perifocal frame back to ECI:

        \[ \mathbf{F_{ECI}} = A_1^T(\Omega) A_2^T(i) A_3^T(\omega) \mathbf{F_{perifocal}} \]

Ephemeris: Position and Velocity Vector as Function of Time#

  • Procedure:

    1. Compute the mean anomaly \(M = n(t - t_0) + M_0\).

    2. Solve Kepler’s equation \(M = E - e\sin(E)\) for \(E\) (the eccentric anomaly).

    3. Find the true anomaly \(\nu\) from:

      \[ \tan \frac{\nu}{2} = \sqrt{\frac{1+e}{1-e}} \tan \frac{E}{2} \]

    4. Compute the position vector in the perifocal frame:

      \[\begin{split} \mathbf{r_{perifocal}} = \frac{a(1 - e^2)}{1 + e \cos \nu} \begin{bmatrix} \cos \nu \\ \sin \nu \\ 0 \end{bmatrix} \end{split}\]

    5. Compute the velocity vector in the perifocal frame:

      \[\begin{split} \mathbf{v_{perifocal}} = \sqrt{\frac{\mu}{a(1 - e^2)}} \begin{bmatrix} -\sin \nu \\ e + \cos \nu \\ 0 \end{bmatrix} \end{split}\]

    6. Transform the position and velocity vectors from the perifocal frame to the ECI frame using the inverse rotation matrices:

      \[ \mathbf{r_{ECI}} = A_3(\omega) A_2(i) A_1(\Omega) \mathbf{r_{perifocal}} \]
      \[ \mathbf{v_{ECI}} = A_3(\omega) A_2(i) A_1(\Omega) \mathbf{v_{perifocal}} \]
